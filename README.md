# Simulador de Identidade Azure + AWS

Este repositório contém um simulador interativo para praticar a criação de usuários, grupos e permissões em **Microsoft Azure AD** e **AWS IAM**. Ele é totalmente **educacional**, sem impactar sistemas reais, ideal para estudo e prática.

---

## Funcionalidades

### Azure AD (Microsoft Entra ID)
- Criar usuário
- Escolher função (Role)
- Adicionar usuário a grupo
- Feedback visual interativo

### AWS IAM
- Criar usuário
- Atribuir a grupo
- Definir política (Policy)
- Feedback visual interativo

---

## Como usar
1. Clone o repositório ou baixe o arquivo `index.html`.
2. Abra o arquivo `index.html` no navegador.
3. Escolha o ambiente que deseja simular: Azure ou AWS.
4. Preencha os campos e clique em **Criar Usuário**.

## Publicação no GitHub Pages
1. Vá em **Settings > Pages**.
2. Selecione a branch `main` e `/ (root)` como source.
3. Clique em **Save**.
4. Após alguns segundos, seu simulador estará disponível online.

Exemplo de URL:
```
https://seu-usuario.github.io/simulador-identidade-azure-aws/
```

## Aviso
Este simulador é apenas **educacional**. Nenhuma ação real em Azure ou AWS é realizada. Não insira credenciais verdadeiras.
